# Example API Tests

## Requirements: 
1. Java 8 or newer installed
2. Maven
3. Allure

## How to run project

1. Run JUnit 5 tests using Maven: 
```bash
mvn test
mvn clean test
```

2. Run allure
```bash
mvn allure:serve
```
3. After generating report please go to the site and change the language :)
